# pixavatar

[![Gem Version](https://badge.fury.io/rb/pixavatar.svg)](https://badge.fury.io/rb/pixavatar)

generate unique pixel avatar for users

## How to use

- IRB

```bash
$ gem install pixavatar
$ irb
> require 'pixavatar'
#=> true
> Pixavatar.draw_image('hello')
#=> generates and saves image on $PWD
```
